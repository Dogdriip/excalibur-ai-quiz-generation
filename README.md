# Quiz Generation

독드립은 퇴근하고 싶다

# Usage

## Requirements

```sh
$ pip install -r requirements.txt
$ sudo apt update && sudo apt install default-jre default-jdk -y
```

## Run

```sh
$ python3 main.py ./sample.pdf
```

## Output

Output goes to stdout as following format: `<QUIZ>|<ANSWER>`.
